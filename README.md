# eWay-CRM ESLint Rules for TypeScript

## Installation

```
npm i --save-dev @eway-crm/eslint-config
```

## Configuration

Add configuration to your ```.eslintrc.json``` project file:

```
{
  "extends": ["@eway-crm/eslint-config"]
}
```
